# Code of Conduct

Technical discussion is welcome when it is useful and respectful.

## Expected Behavior

- be specific
- be respectful
- focus on actionable technical value

## Unacceptable Behavior

- insults, mockery, or humiliation
- hostile or baiting comments
- low-effort criticism with no useful direction
- spam or repeated non-contributive behavior

## Enforcement

The repository owner may remove, ignore, lock, or report content that does not help improve the repository.
